MVP Scope (Phase 1)

Core functionality for initial release

FR-1

Google Calendar Connection

  • OAuth 2.0 authentication flow
  • Support for Workspace and Gmail
  • Clear permission scopes
  • Easy disconnect option

FR-2

One-Way Sync

  • Deadlines create calendar events
  • Events include title, project, link
  • Auto-update on deadline change
  • Auto-delete on completion

FR-3

Sync Configuration

  • Global sync on/off toggle
  • Filter by project
  • Filter by priority level
  • Filter by task assignment

FR-4

Calendar Selection

  • Choose target calendar
  • Create dedicated TaskFlow calendar
  • Support multiple calendars
